WriteOnlyMemory
Der Chip-Hersteller Signetics brachte 1973 ein Datenblatt für ein Write-Only-Memory heraus.

Unter https://github.com/spacerace/write-only-memory habe ich einen Treiber für diesen Chip veröffentlicht.

So wird er übersetzt und geladen:
$ apt-get install linux-headers- build-essential git
$ git clone https://github.com/spacerace/write-only-memory
$ cd write-only-memory
$ make
$ insmod wom.ko
$ dmesg | tail	# look for major number, see one line down
$ mknod /dev/wom c 248 0
Danach kann er benutzt werden:
# cat /dev/wom 
write only memory does not support read operations. Already wrote 0bytes to nowhere.
# cat /dev/urandom > /dev/wom 
^C
# cat /dev/wom 
write only memory does not support read operations. Already wrote 207749120bytes to nowhere.
Ein paar Sekunden nach dem "cat" drückte ich STRG-C um abzubrechen. Bis dahin schrieb er ca. 200MB nach nirgendwo.
